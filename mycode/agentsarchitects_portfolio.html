Automatic Speech Recognition (ASR) – Model Fine-Tuning & Optimization

Deep Learning | Streaming ASR | Model Fine-Tuning | Production-Ready Speech Systems
This project focused on building and fine-tuning a high-accuracy Automatic Speech Recognition (ASR) system using NVIDIA NeMo. The implementation covered both CTC-based and Transducer (RNNT) architectures to support high-accuracy transcription and low-latency streaming speech recognition use cases. The system includes a complete end-to-end ASR pipeline — from dataset preparation, audio preprocessing, and custom manifest generation to SentencePiece BPE tokenizer training and model fine-tuning on the LibriSpeech train-clean-100 dataset. Evaluations were conducted on dev-clean and dev-other to ensure robustness across clean and noisy speech conditions. The project demonstrates deep expertise in model architecture tuning, GPU optimization, and deployment-ready speech systems.

Key Metrics

  • FastConformer-CTC: ~6.1% WER on dev-other
  • FastConformer-RNNT: 3.44% WER (dev-clean)
  • FastConformer-RNNT: 3.61% WER (dev-other)
  • Streaming ASR support with low-latency inference
  • Robust performance on noisy & real-world speech
  • Custom BPE tokenizer & decoder resizing
  • bf16 mixed precision GPU training optimization
  • Checkpointing & resume-training support

Technology Stack & Model Architecture

  • Python for model training & experimentation
  • PyTorch for deep learning model development
  • NVIDIA NeMo for ASR pipeline implementation
  • FastConformer (CTC & RNNT) architectures
  • SentencePiece BPE tokenizer training
  • HuggingFace Datasets for LibriSpeech integration
  • LibriSpeech dataset (train-clean-100)
  • TensorBoard for experiment tracking
  • Mixed Precision (bf16) GPU acceleration
  • Model export to .nemo for deployment readiness
